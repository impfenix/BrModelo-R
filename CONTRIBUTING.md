# Contribuindo para o BrModelo F

Obrigado pelo seu interesse em contribuir para o BrModelo F! Este documento fornece diretrizes para contribuir com este projeto.

## Como Contribuir

### Reportando Bugs
- Verifique se o bug já foi reportado nas [Issues](https://github.com/seu-usuario/brmodelo-f/issues).
- Se não, abra uma nova issue descrevendo o problema, passos para reproduzir e informações sobre seu ambiente.

### Sugerindo Funcionalidades
- Abra uma nova issue com a tag `enhancement`.
- Descreva detalhadamente a funcionalidade e por que ela seria útil.

### Pull Requests
1. Faça um fork do repositório.
2. Crie uma branch para sua funcionalidade (`git checkout -b feature/minha-funcionalidade`).
3. Faça o commit de suas alterações (`git commit -m 'Adiciona minha funcionalidade'`).
4. Faça o push para a branch (`git push origin feature/minha-funcionalidade`).
5. Abra um Pull Request.

## Estilo de Código
- Siga as convenções de TypeScript e React.
- Use nomes de variáveis descritivos.
- Comente o código onde for necessário.

## Processo de Desenvolvimento
- Execute `npm install` para instalar as dependências.
- Execute `npm run dev` para iniciar o servidor de desenvolvimento.
- Antes de enviar um PR, execute `npm run lint` para verificar erros de tipo.
